# Diamond Certificate DApp

A blockchain-based decentralized application (DApp) for verifying diamond certificate authenticity using Ethereum smart contracts.

## Features
- Verify gemstone authenticity using serial-number lookup
- Tamper-resistant certificate storage on blockchain
- Transparent and decentralized verification system

## Technology Stack
- Solidity
- Ethereum (Sepolia Testnet)
- Remix IDE
- MetaMask

## Smart Contracts
- diamond_parent.sol
- diamond_child.sol

## Purpose
This project demonstrates the use of blockchain technology to create a tamper-proof system for verifying diamond certificates and preventing certificate fraud.
